# DesignPatternsApp
📘 Design Patterns Java Project
📌 Overview : This project is a simple Notification System developed using Java to demonstrate design patterns and OOP concepts.

🧩 Patterns Used : Observer Pattern , Factory Pattern

💻 Files : Observer.java , User.java , Channel.java , UserFactory.java , Main.java

▶️ How to Run : Open project in NetBeans , Run Main.java

📤 Output : Alice received: New Video Uploaded! , Bob received: New Video Uploaded!
